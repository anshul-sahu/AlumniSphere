🎓 AlumniSphere (Alumni Management System)
A full-stack Alumni Management System built using Spring Boot and React, enabling institutions to manage alumni data, mentorship, events, and job opportunities with admin approval workflows.

📌 Features
👤 User Management
User Registration & Login
Role-based access (Admin, Alumni, Student)
Account approval system (Admin controlled)
✅ Admin Panel
Approve / Reject users
Manage alumni data
Monitor system activities
🎓 Alumni Module
Maintain professional profiles
Share career updates
Provide mentorship
🤝 Mentorship System
Alumni can offer mentorship
Students can request guidance
💼 Job & Internship Portal
Alumni can post job opportunities
Students can explore openings

💻 Tech Stack
🔹 Frontend
React.js
HTML, CSS, JavaScript, Tailwind CSS
🔹 Backend
Spring Boot
Spring Data JPA (Hibernate)
REST APIs
🔹 Database
MySQL
🔹 Tools
Git & GitHub
Postman
Maven
🔹 Deployment
AWS EC2
